## Script usage

```
  k8s-service {start/stop/status}
```

Deployment path :   /opt/SDSW/scripts 

The output log will be in the same path :  /opt/SDSW/scripts

Output  will be: 
  1  for Node Ready
  0  for Node Drained

For other cases check the output log file
